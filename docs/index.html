¿Qué es Meowndloads?

Meowndloads es un pequeño software desarrollado en python con GTK+3 para descargar música, videos y playlists desde diferente tipo de servidores, utilizando la librería en python llamada "Youtube-dl"

Construido en Colombia, se espera que éste cumpla con sus espectativas y que la comunidad nos ayude a mejorar y ampliar sus funcionalidades, para desarrollar el mejor software de descarga de multimedia.

Si eres desarrollador y te interesa trabajar en este proyecto, te invito a que contactes con los desarrolladores y te unas a este equipo. Si eres diseñador y estas dispuesto a colaborar al equipo con el logo de la aplicación, se te agradecería muchísimo.

Si deseas conocer más sobre youtube-dl te invitamos a ver esta página web

Ejecución

Dependencias

Este software requiere de las siguientes dependencias para funcionar en un 100%, investiga un poco el funcionamiento de la instalación de estos paquetes según tu distribución.
Aún se encuentra en estado Beta por lo que los archivos de instalación aún no están disponibles.

  • Python 3
  • PyGTK >= 3.18
  • Notify
  • urllib (preinstalada por defecto en python3)
  • youtube-dl (pip3 install youtube_dl)

Ejecución Manual desde GitHub

Se requiere tener instalado Git en su distribución. 

	$ git clone https://github.com/WilferCiro/meowndloads.git
	$ cd meowndloads
	$ python3 meowndloads_app/window_multi.py

Ejecución Manual desde código tar.xz

Ejecuta el programa desde el archivo fuente tar.xz presente en esta página, para ello debes tener instalado wget. 

	$ wget https://wilferciro.github.io/meowndloads/aplicacion_alojamiento/meowndloadsv0.1.1.tar.xz
	$ tar -xvf meowndloadsv0.1.1.tar.xz
	$ cd meowndloads
	$ python3 meowndloads_app/window_multi.py
